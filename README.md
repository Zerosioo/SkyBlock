# Skyblock plugin
Author: Zerosio

Note: This is just a prototype.

## Available commands:
- `/rank <rank_name> <player>` (if player name not specified gets executor)
- `/warp <warp>`

## Features:
- Ranks System
- World based server system
- Profiles System
- Scoreboard
- User economy
- Instaces with auto tp to instance on join.
- Stats system
